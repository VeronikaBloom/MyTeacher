О системе "МойПедагог"

Назначение системы

Информационная система «МойПедагог» разработана для автоматизации взаимодействия между преподавателями и студентами ГАПОУ КК ЛСПК (Ленинградского социально-педагогического колледжа).

Она позволяет:

  • преподавателям заполнять и редактировать анкеты, добавлять информацию о себе и своем расписании;
  • студентам — просматривать анкеты преподавателей и их актуальное расписание занятий.

Функции и возможности

Для преподавателей:

  1. Регистрация по роли «Преподаватель»
  2. Заполнение анкеты: ФИО, должность, опыт работы, квалификация и т.д.
  3. Добавление и редактирование расписания
  4. Редактирование профиля в любое время

Для студентов:

  1. Авторизация по роли «Студент»
  2. Просмотр списка преподавателей
  3. Переход к анкетам и расписаниям преподавателей
  4. Удобный интерфейс для поиска информации

Для администратора:

  1. Полный доступ ко всем анкетам
  2. Возможность редактирования и удаления данных
  3. Контроль корректности заполнения

Особенности реализации

  • Адаптивный интерфейс — система удобна для использования как на компьютерах, так и на смартфонах
  • Без использования серверных языков (PHP) и баз данных — все данные сохраняются в localStorage
  • Простая регистрация по ролям
  • Фильтрация отображения — в расписании и списке анкет отображаются только заполненные строки
